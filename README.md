# LinkCheck AI

LinkCheck AI is a Flask web app that analyzes URLs for phishing and malicious indicators using:

- a trained scikit-learn model,
- URL and HTML heuristics,
- SSL/DNS checks,
- external threat intelligence APIs,
- optional screenshots for suspicious URLs,
- optional Discord feedback reporting.

## Features

- Web UI for URL analysis
- `POST /analyze` JSON API
- ML + heuristic phishing score
- VirusTotal, Google Safe Browsing, IPQualityScore, URLhaus, and PhishTank integration
- Tranco reputation support when `data/tranco_GV97K-1m.csv.zip` is present
- Async screenshots for suspect/dangerous URLs
- Feedback storage and optional Discord webhook

## Local Setup

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
copy .env.example .env
python main.py
```

Open:

```text
http://127.0.0.1:5000
```

## Environment Variables

Create `.env` from `.env.example`, then fill the keys you have:

```env
VIRUSTOTAL_API_KEY=
GOOGLE_SAFE_BROWSING_API_KEY=
IPQS_API_KEY=
URLHAUS_AUTH_KEY=
PHISHTANK_API_KEY=
DISCORD_FEEDBACK_WEBHOOK=
FEEDBACK_ADMIN_TOKEN=
```

Leave a key empty to skip that provider. Do not commit `.env`.

## API

### Analyze A URL

```bash
curl -X POST http://127.0.0.1:5000/analyze ^
  -H "Content-Type: application/json" ^
  -d "{\"url\":\"https://example.com\"}"
```

Response includes:

- `score`
- `verdict`: `safe`, `suspect`, `dangerous`, or `error`
- `reasons`
- `ml_score`
- `heuristic_score`
- `trust_score`
- `threat_intel`

### Feedback Admin

If `FEEDBACK_ADMIN_TOKEN` is configured:

```text
/admin/feedback?token=YOUR_TOKEN
```

## Deploy On Render

This repo includes `render.yaml` and a Dockerfile.

1. Push the repo to GitHub.
2. In Render, create a new Blueprint or Web Service from the GitHub repo.
3. Use Docker environment.
4. Add these secret environment variables in Render:

```env
VIRUSTOTAL_API_KEY=
GOOGLE_SAFE_BROWSING_API_KEY=
IPQS_API_KEY=
URLHAUS_AUTH_KEY=
PHISHTANK_API_KEY=
DISCORD_FEEDBACK_WEBHOOK=
```

`SECRET_KEY` and `FEEDBACK_ADMIN_TOKEN` can be generated by `render.yaml`.

Render provides `PORT` automatically. The Docker command binds to `${PORT:-5000}`.

## Docker

```bash
docker build -t linkcheck-ai .
docker run --env-file .env -p 5000:5000 linkcheck-ai
```

## Security Notes

- Never commit `.env`.
- Rotate any API key that was accidentally pushed.
- Discord webhook URLs are secrets.
- This project is a detection aid, not a replacement for a professional security gateway.
